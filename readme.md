### 🍎 How to Contribute

1. 각 분야 별 새 디렉터리 만들기 ( ex. 그리디, 구현 ... )
2. 백준 문제는 해당 디렉터리 내부에서 새로 디렉터리 만들고 문제 추가하기 ( ex. bj_greedy/bj_1000 .. )
3. 파일 이름 : 문제 번호 통일하기

**📌 Push 규칙**  
-> pull 부터

```
$ git pull <remote 이름> master
```

-> **분야/n주차/파일명** 형식으로 commit & push

```
$ git add .
$ git commit -m "greedy/1주차/01.py"
$ git push <remote 이름> master
```
